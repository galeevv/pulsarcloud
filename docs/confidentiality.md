Политика конфиденциальности
Дата последнего обновления: 20 июля 2026 г.

1. Общие положения
Настоящая Политика определяет порядок обработки и защиты персональных данных пользователей пользователей сервиса Pulsar. Она разработана в соответствии с Федеральным законом РФ № 152-ФЗ «О персональных данных» и иными нормативными актами Российской Федерации.
Используя сервис, пользователь подтверждает согласие с условиями Политики. При несогласии необходимо прекратить использование сервиса.

2. Персональные данные, собираемые сервисом
Сервис придерживается принципа минимального сбора данных.
Могут обрабатываться следующие данные:
2.1 username (при регистрации);
2.2 технические данные аккаунта (ID пользователя, дата регистрации, статус подписки);
2.3 платежная информация, необходимая для обработки оплаты (без хранения данных банковских карт).
Сервис не ведёт журналы активности, не собирает IP-адреса, историю подключений и передаваемый трафик.

3. Цели обработки данных
Персональные данные используются исключительно для:
3.1 регистрации и идентификации пользователя;
3.2 предоставления доступа к VPN-услугам;
3.3 обработки платежей;
3.4 работы личного кабинета;
3.5 предоставления технической поддержки;
3.6 исполнения договорных обязательств.

4. Хранение и защита данных
Данные пользователей хранятся в защищённой инфраструктуре сервиса.
Доступ к данным имеют только уполномоченные лица, необходимые для обеспечения работы сервиса.
В случае удаления аккаунта пользователя его данные удаляются в течение 30 календарных дней, если иное не требуется законодательством.

5. Передача данных третьим лицам
Персональные данные не передаются третьим лицам, за исключением случаев, предусмотренных законодательством Российской Федерации. В требуемых законом ситуациях передача осуществляется только на основании законного запроса уполномоченных органов.

6. Права пользователя
Пользователь имеет право:
получать информацию об обработке своих персональных данных;
требовать уточнения, блокирования или удаления данных;
отзывать согласие на обработку персональных данных.

7. Контактная информация
По вопросам обработки персональных данных пользователь может обратиться в службу поддержки через систему поддержки внутри сервиса.
